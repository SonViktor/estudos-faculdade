## Operadores ##

**Operadores Aritméticos**

- Soma. Símbulo: +
- Subtração. Símbulo -
- Multiplicação. Símbolo *
- Divisão. Símbolo /
- Modulo. Resto de divisão inteira. Simbolo: %

- Prioridades, parênteses. Símbolo: ()

**Ex. Uso de Operadores Aritméticos**

int A = 2, B = 3, resultado = 0;

resultado = A + B;
resultado = A - B;
resultado = A * b;
resultado = (A + 4)/3;

**Operadores Relacionais**

- Maior que. Símbolo: >
- Maior ou igual a. Símbolo: >=
- Menor que. Símbolo: < 
- Menor ou igual a. Símbolo: <=

- Igual. Símbolo: ==
- Diferente. Símbolo: !=

A = 2
B = 3

if (A > B)

if (A <= 3.14)

if (B != 10)

if ((A - B) == 0)

**Operadores Logícos**

- Negação
inverte um valor lógico. Símbolo: !

- Conjunção 
Operação "E" logíca. Símbolo: &&

- Disjunção
Operação "OU" lógica. Símbolo ||

A = 2
B = 3

if (A < 10 && A > 0)

if (A >= 0 || B >= 0)

if (!(A == B))
